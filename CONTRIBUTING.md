# 🤝 Contributing to RMAG-2026

Thanks for being interested! We welcome contributions of all kinds—not just code.

---

## 🎯 What We're Looking For

### 💬 Ideas & Feedback (No coding required!)
- Found a typo? Let us know via [issues](../../issues)
- Want to suggest a new prompt technique? [Open an issue](../../issues)
- Think the README is confusing? Tell us what's unclear

### 🐛 Bug Reports
- App not working? [Report it](../../issues)
- Data looks wrong? Help us fix it

### 📝 Documentation
- Improve explanations in README
- Add examples to prompts
- Write tutorials or guides

### 🔬 New Experiments
- Building something in geoscience + AI? We'd love to add it
- Have an interesting dataset or analysis? Share it

---

## 🚀 How to Get Started

### **For typo fixes & small docs edits:**
1. Click the file you want to edit
2. Hit the pencil icon (✏️) in GitHub
3. Make your change
4. Click "Commit changes" → "Create a new branch for this commit"
5. We'll review & merge it!

### **For bigger changes (new prompts, bug fixes, new features):**
1. **Fork** this repo
2. **Create a branch:** `git checkout -b my-improvement`
3. **Make your changes**
4. **Push to your fork:** `git push origin my-improvement`
5. **Open a Pull Request** with a clear description of what you changed & why
6. We'll review and discuss!

---

## 📋 Before You Contribute

- **Keep it respectful & on-topic** (focus on geoscience, AI, or critical thinking)
- **If adding a new prompt:** Include a one-line description and a use case
- **If reporting a bug:** Tell us:
  - What you did
  - What you expected to happen
  - What actually happened
  - What browser/device you used

## ❓ Questions?

- **"Can I use this in my company?"** → Check the [LICENSE](./LICENSE)
- **"Do I need permission to fork?"** → Nope—that's the whole point of open source!
